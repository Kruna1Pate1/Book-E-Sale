## Book-E-Sale :books:

E-Commerce platform for buying & selling of books.

### Technologies

#### Frontend
- TypeScript
- ReactJS
- React Router
- Styled Component
- Toastify
- Formik and Yup

#### Backend
- Java
- Spring Boot
- Spring Security
- JWT Token for Authentication and Authorization

#### Database
- PostgreSQL

### Modules
- Login
- Register
- Add Book
- Book Listing
- Cart
